# 纯前端一键抠图工具

一个完全在浏览器端运行的背景移除工具，无需上传图片到任何服务器。

## 特点

- 纯前端实现，所有处理都在用户浏览器中完成
- 隐私安全，图片不会上传到任何服务器
- 易于部署，只需托管静态文件
- 响应式设计，支持各种设备

## 技术栈

- HTML5
- CSS3
- JavaScript
- [rembg-browser](https://github.com/cubiq/rembg-browser) - 浏览器端背景移除库

## 部署

此项目可以部署到任何静态文件托管服务：

- GitHub Pages
- Vercel
- Netlify
- 或任何支持静态文件托管的服务

## 使用方法

1. 访问部署后的网站
2. 上传或拖拽图片到上传区域
3. 点击"一键移除背景"按钮
4. 等待处理完成后下载结果

## 注意事项

- 首次处理可能需要下载模型文件，后续处理会更快
- 对于较大的图片，处理时间可能会较长
- 所有处理均在浏览器中完成，不依赖任何后端服务
